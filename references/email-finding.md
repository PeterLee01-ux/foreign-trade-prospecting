# 邮箱查找工作流  ## 适用场景 用户需要查找特定公司联系人的邮箱地址，或在目标行业/地区内寻找潜在客户。  ## 重要规则  1. **每次调用 Snov.io 查找邮箱前，必须停下来让用户输入模糊查找关键词**，等用户确认后再继续。不得自动生成关键词。 2. **每个公司最多查 4 个邮箱**，严格控制 Snov.io 点数消耗。  ## 预设职位查找关键词  使用预设关键词列表（见 SKILL.md 全局规则），模糊匹配 job_position。 默认包含：president, chief, chair, director, GM, partner, CEO, head, procurement, purchas, sourc, buyer, vendor, supplier, supply。  用户可在暂停点增删修改。  ## 两种运行模式  | 模式 | 工具 | 适用场景 | |------|------|---------| | **MCP 模式** | Snov.io MCP + Prospector MCP | Codex 内直接调用 | | **免 MCP 模式** | `python scripts/snovio_api.py` | 无法配 MCP，命令行直调 |  两种模式的效果相同，免 MCP 模式需要先 export 密钥： ```bash export SNOVIO_CLIENT_ID=xxx export SNOVIO_CLIENT_SECRET=xxx ```  ## 操作步骤（MCP 模式）  > [EXCEPTIONS] 完整异常处理见 flow-exceptions.md 阶段 1+2  ### 第一步：明确目标 与用户确认： - 目标公司域名还是行业关键词？ - 需要对接什么职位？（默认使用预设关键词，可增删） - 国家/地区范围？ - 预计需要多少家公司？  ### 第二步：域名搜索 使用 Snov.io domain_search 查找公司域名。  免 MCP 备选：`python scripts/snovio_api.py domain-search "LED lighting California"`  ### 第三步：发现域名下的联系人  > [EXCEPTIONS] 如果 domain_search 零结果：换更宽泛的英文关键词重试（如 "LED" 替代 "LED driver"），或手动输入已知公司域名。  > [EXCEPTIONS] 如果 domain_prospects 零结果：该公司无匹配联系人，跳过此公司或换更宽关键词。  > [PHONE] Snov.io company_info 可能包含 phone 字段，可用于后续 WhatsApp 验证。 使用 Snov.io domain_prospects。  免 MCP 备选：`python scripts/snovio_api.py domain-prospects acme.com --job procurement`  ### 第四步：【暂停点】确认查找关键词  **⛔ PAUSE — NEVER proceed without user confirmation.**，向用户展示： - 当前目标公司列表 - 当前使用的职位关键词列表 - 询问用户：是否使用预设关键词继续，还是需要增删修改？  ### 第五步：查找邮箱地址 使用 Snov.io find_email。每公司最多 4 个邮箱。  免 MCP 备选：`python scripts/snovio_api.py find-email --first John --last Smith --domain acme.com`  ### 第六步：备选 — Prospector 免费查找 当 Snov.io 额度不足时，用 Prospector（免费 50 次/天）。  ### 第七步：验证邮箱 跳转到 email-verification.md。  ## 操作步骤（免 MCP 模式）  如果 MCP 不可用，直接用 Python 脚本：  ```bash # 1. 域名搜索 python scripts/snovio_api.py domain-search "scent diffuser"  # 2. 查域名下联系人 python scripts/snovio_api.py domain-prospects target-domain.com --job procurement --limit 20  # 3. 找具体邮箱 python scripts/snovio_api.py find-email --first John --last Smith --domain acme.com  # 4. 查公司详情 python scripts/snovio_api.py company-info acme.com  # 5. 随时查余额 python scripts/snovio_api.py balance ```  ## 常见错误 & 自救  | 错误 | 原因 | 解决 | |------|------|------| | SNOVIO_CLIENT_ID not set | 免 MCP 模式密钥未配 | export 密钥后重试 | | Snov.io 403 | API Key 无效或过期 | 让用户去 Snov.io 重新获取 | | Snov.io 429 | 请求频率超限 | 等待 60 秒后重试 | | domain-search 无结果 | 关键词太窄或拼写不对 | 用更宽泛的英文关键词，或直接搜公司名 | | find_email 无结果 | 该人在 Snov.io 数据库中没有记录 | 换 LinkedIn 手动找，或用 Prospector 备选 | | domain_prospects 无结果 | 公司太小或无公开联系人 | 换更大公司，或 LinkedIn 手动查找 | | Prospector 报错 | Node.js 未安装或版本 < 18 | `node --version`，需 >=18 | | 信用耗尽 | Snov.io 余额不足 | 降级用 Prospector（免费 50/天）或手动搜索 |
